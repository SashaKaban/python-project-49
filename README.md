### Description

This repository contains 5 games:
    brain-calc
    brain-even
    brain-prime
    brain-progression
    brain-gcd
Answer to 3 questions to win or try again if you fail!

### Setup

```bash
make install
```

### Run example

```bash
brain-even
```
